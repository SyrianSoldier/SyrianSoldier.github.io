                # 定积分定义式

                ∫abf(x)dx=lim⁡λ→0∑i=1nf(ξi)Δxi \int_{ a}^{b} f(x)dx = \lim_{ \lambda \to 0}{ \sum_{ i=1}^{n}f(\xi_i)\Delta x_i } ∫ab​f(x)dx=λ→0lim​i=1∑n​f(ξi​)Δxi​

                1. 分割: 将曲边梯形分割成iii个
                  • f(ξi)代表第i个区间的高f(\xi_i)代表第i个区间的高f(ξi​)代表第i个区间的高
                  • Δxi代表第i个区间的宽\Delta x_i 代表第i个区间的宽Δxi​代表第i个区间的宽
                2. 近似: 每个区间的面积近似为 $\Delta S_i ≈ f(\xi_i)\Delta x_i $
                3. 求和: $曲边梯形面积等于每个小区间面积的和, 即S_{曲边梯形} = \sum_{ i=1 }^{n } \Delta S_i $
                4. 取极限: ∫abf(x)dx=lim⁡λ→0∑i=1nf(ξi)Δxi\int_{ a}^{b} f(x)dx = \lim_{ \lambda \to 0}{ \sum_{ i=1}^{n}f(\xi_i)\Delta x_i }∫ab​f(x)dx=limλ→0​∑i=1n​f(ξi​)Δxi​
                  • λ=max⁡(Δx1,Δx2,...,Δxi),最宽的区间为λ\lambda=\max(\Delta x_1,\Delta x_2,...,\Delta x_i), 最宽的区间为\lambdaλ=max(Δx1​,Δx2​,...,Δxi​),最宽的区间为λ
