# 读书的三个阶段

1. 第一阶段：理解70%作者的话 
2. 第二阶段: 练习 
3. 第三阶段: 做思维导图(整理知识)

例:
1. 数学
    - 听课能听懂70%就行
    - 做练习题
    - 做思维导图
数学需要听一章的课, 做一章的练习题, 做一章的思维导图

2. 数据结构
    - 听咸鱼视频能听懂70%就行
    - 做练习题
    - 做思维导图


4. 计算机网络
    - 看王道课本, 能看懂70%就行
    - 做练习题
    - 做思维导图


5. vue设计与实现
   - 看书, 理解70%就行
   - 做练习题
   - 做思维导图
